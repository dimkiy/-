# Начало
Обход белого списка для тех, кто живёт в РФ. Прошу обратить внимание на то, что автор репозитория ни как не относится к команде разработчиков обхода. Я лишь хочу поделится тем, как обойти белые списки.

# ШАГ 1
И так, чтобы обойти систему БЕЛЫХ СПИСКОВ нам понадобится Telegram. Там мы должны будем найти этого бота: @VetryxVPN_bot. Хочу заметить, что бот БЕСПЛАТНО работает только 24 часа. Дальше только платно.
# ШАГ 2
После того, как мы запустили бота (по команде /start), нам нужно будет зайти в раздел "Подписка". Там в сообщении бота будет дана ВАША личная ссылка для подключения к обходу. Эту ссылку мы копируем и переходим в приложение Happ. Далее нажимаем в Happ на плюсик, который находится справа сверху и выбираем "вставить из буфера обмена".
# ШАГ 3
Всё! Осталось только запустить Анти-Глушилку #1,2,3 или 4. С данным обходом работают пока: Сбер Банк, Тинькофф, Госуслуги, Тик Ток, Вк, Телеграм (не всегда), ВСЕ зарубежные нейросети, Пинтерест и многое другое!
